# Autonomous Personal AGI System

Production-grade Python scaffold for a continuously running modular multi-agent AGI.

## Quick start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
python -m autonomous_agi.main --config config/system.yaml
```

## Architecture

- Orchestrator Agent
- Memory Agent
- Web Agent
- Communication Agent
- Scheduler Agent
- Device Monitor Agent
- Code Agent
- Self-Reflection Agent
- Security Agent
- Tool Agent

See `examples/use_cases.md` for scenario flows.
